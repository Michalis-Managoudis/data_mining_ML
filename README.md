# data_mining_ML

## 1. Health dataset

Στο dataset που περιέχει πληροφορίες ασθενών πραγματοποιείται ανάλυση δεδομένων, χειρισμός τιμών, και πρόβλεψη μεταβλητής (Random Forest).
